Loops let you execute code repeatedly. C has three loop constructs, each suited to different patterns.

## The for Loop

Use `for` when you know how many times to repeat:

<Editor defaultCode={`#include <stdio.h>\n\nint main() {\n    for (int i = 0; i < 5; i++) {\n        printf("Iteration %d\\n", i);\n    }\n    \n    return 0;\n}\n`} time="O(n)" space="O(1)" />

The `for` statement has three parts separated by semicolons:

```c
for (initialization; condition; update) {
    // body
}
```

1. **Initialization** — runs once before the loop starts (`int i = 0`)
2. **Condition** — checked before each iteration; loop stops when false (`i < 5`)
3. **Update** — runs after each iteration (`i++`)

## The while Loop

Use `while` when you don't know the count in advance:

<Editor defaultCode={`#include <stdio.h>\n\nint main() {\n    int n = 12345;\n    int digits = 0;\n    \n    while (n > 0) {\n        n /= 10;    // Remove last digit\n        digits++;\n    }\n    \n    printf("Number of digits: %d\\n", digits);\n    \n    return 0;\n}\n`} time="O(log n)" space="O(1)" />

The `while` loop checks the condition **before** each iteration. If the condition is false from the start, the body never executes.

## The do-while Loop

Guarantees the body runs **at least once**:

<Editor defaultCode={`#include <stdio.h>\n\nint main() {\n    int n;\n    \n    do {\n        printf("Enter a positive number: ");\n        scanf("%d", &n);\n    } while (n <= 0);\n    \n    printf("You entered: %d\\n", n);\n    \n    return 0;\n}\n`} stdin="5" time="O(1)" space="O(1)" />

<Callout type="info">
`do-while` is less common than `for` and `while`, but it's perfect for input validation: you want to read at least once, then check if the input is valid.
</Callout>

## break and continue

Two keywords control loop execution:

<Editor defaultCode={`#include <stdio.h>\n\nint main() {\n    // break: exit the loop entirely\n    printf("break example:\\n");\n    for (int i = 0; i < 10; i++) {\n        if (i == 5) break;\n        printf("%d ", i);\n    }\n    printf("\\n\\n");\n    \n    // continue: skip to the next iteration\n    printf("continue example:\\n");\n    for (int i = 0; i < 10; i++) {\n        if (i % 3 == 0) continue;\n        printf("%d ", i);\n    }\n    printf("\\n");\n    \n    return 0;\n}\n`} time="O(n)" space="O(1)" />

- **`break`** — immediately exits the innermost loop
- **`continue`** — skips the rest of the current iteration and jumps to the next one

## Nested Loops

Loops can be placed inside other loops. The inner loop runs completely for each iteration of the outer loop:

<Editor defaultCode={`#include <stdio.h>\n\nint main() {\n    // Multiplication table (1-5)\n    for (int i = 1; i <= 5; i++) {\n        for (int j = 1; j <= 5; j++) {\n            printf("%4d", i * j);\n        }\n        printf("\\n");\n    }\n    \n    return 0;\n}\n`} time="O(n²)" space="O(1)" />

<Callout type="info">
`%4d` is a format specifier that prints an integer right-aligned in a field of 4 characters wide. This keeps the columns aligned.
</Callout>

## Infinite Loops

Sometimes you want a loop that runs forever (until explicitly stopped):

```c
while (1) {
    // runs forever
    // use break to exit
}

for (;;) {
    // also runs forever — all three parts of for are optional
}
```

## Common Patterns

### Counting up
```c
for (int i = 0; i < n; i++) { ... }
```

### Counting down
```c
for (int i = n - 1; i >= 0; i--) { ... }
```

### Reading until end of input
```c
int n;
while (scanf("%d", &n) == 1) { ... }
```

### Accumulating a sum
```c
int sum = 0;
for (int i = 1; i <= n; i++) {
    sum += i;
}
```
